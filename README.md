# Car Price Prediction 🚗

## Objective
Predict used car prices using Machine Learning.

## Method
- PCA (Dimensionality Reduction)
- Linear Regression

## Features Used
- Year
- Present Price
- Kms Driven
- Fuel Type
- Transmission

## Tools
- Python
- Pandas
- Scikit-learn

## Output
Predicts selling price of cars.
